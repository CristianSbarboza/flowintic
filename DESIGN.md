# Design System: Flow In Tic

## 1. Visual Theme & Atmosphere
**Atmosphere:** Futuristic, Fluid, and High-Performance.
The design philosophy is centered around "Digital Flow." It uses a **Dark Mode** foundation with layered translucency (glassmorphism) to create depth. The vibe is sophisticated yet energetic, moving away from static layouts toward a sense of constant technical motion.

## 2. Color Palette & Roles
*   **Deep Space Black (#050505):** Primary background, providing a canvas for high-contrast elements.
*   **Electric Cyan (#00F2FF):** Primary action color and "flow" indicator. Used for main CTAs and highlights.
*   **Cyber Purple (#9D00FF):** Accent color for secondary buttons and gradient transitions.
*   **Glass White (#FFFFFF10):** Background for containers and cards, using backdrop blur.
*   **Muted Slate (#94A3B8):** Body text for balanced readability.

## 3. Typography Rules
*   **Headings:** `Space Grotesk` (Rationale: Industrial, tech-forward, high impact).
    - Bold weights for headlines.
    - Large scale (H1: 88px) for hero impact.
*   **Body:** `Inter` or `Raleway` (Rationale: Clean, legible, modern).
    - Regular/Medium weights.
    - Letter spacing: -0.01em for a tighter, premium feel.

## 4. Component Stylings
*   **Buttons:**
    - **Primary:** Pill-shaped (`rounded-full`), Electric Cyan background, dark text. Subtle pulse animation on hover.
    - **Secondary:** Transparent with `backdrop-blur`, thin `Cyber Purple` border.
*   **Cards/Containers:**
    - **Glass Cards:** Subtly rounded corners (`rounded-2xl`), semi-transparent background, `1px` border using a subtle linear gradient.
*   **Inputs/Forms:** 
    - Dark background, `1px` Cyan border on focus, minimalist icons.

## 5. Layout Principles
*   **Fluidity:** Use of asymmetric grids and floating elements to break the "boxed" feel.
*   **White Space:** Generous margins between sections to let the "premium" feel breathe.
*   **Transitions:** Smooth opacity and transform transitions (ease-in-out) for all interactive elements.
